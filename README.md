# Ruboty::Redmine

Redmine plugin for Ruboty

## Available commands

```
ruboty create issue "subject" "project name" project "tracker name" tracker
```

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-redmine'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruboty-redmine
